# Software Requirements Specifications

Software requirements are not publicly available, but access to specification requirements on the following topics can be granted upon request:

- Sign in  
- Device Pairing  
- Questionnaires  
- Watch software update  
- System menu (Support, Privacy policy, Terms of use)  
- Log out & Account Deletion  
- Insights Alarms Triggering App Push Notifications  
- Localization  
- Watch Battery Alerts  
- Customization
